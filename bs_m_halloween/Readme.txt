# bs_m_halloween Clothes Add-on - README

### Description:
This is a drag-and-drop custom clothing pack designed for use on FiveM servers. It includes 9 outfits with multiple textures for each item.

### Installation:

1. **Download the resource**: Ensure you've downloaded the `bs_m_halloween` folder from the provided source.
   
2. **Drag and Drop**: Drag and drop the `bs_m_halloween` folder into your server's `resources` directory.

3. **Add to server.cfg**:
   - Open your `server.cfg` file.
   - Add the following line to ensure the resource starts correctly:

     ```
     ensure bs_m_halloween
     ```

4. **Restart the server**: Either restart the server or run `ensure bs_m_halloween` in the console to activate the resource.
